ardyh bots

ardyh bots or just bots, are smart, interactive devices that live in the real-world and connect to ardyh.io. So far bots have been Raspberry Pi's connected to various sensors, motors, and lights that automoaticly connect to aryh using wi-fi. But bots could be made from other electronics like an Arduino for instance. A bots is really a package deal. The pacakge consists of hardware and software. bots need bot client code (code that runs on the Raspberry Pi) and a web or mobile client (primarly used for control and configuration).
